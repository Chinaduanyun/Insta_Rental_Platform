# 一拍即租 - 拍立得相机租赁平台

## 📋 项目概述

**一拍即租（Insta Rental Platform）** 是一个面向年轻用户群体的拍立得相机在线租赁平台，提供多品牌拍立得相机（富士 Instax、宝丽来等）的短期租赁服务。平台采用移动端优先设计（390px × 844px），模拟真实手机端应用体验，支持用户端和商户端双角色操作。

**核心理念：** 让更多人以低成本体验高品质拍立得相机，无需购买即可享受拍照乐趣。

**技术栈：** HTML5 + Tailwind CSS + 原生 JavaScript + localStorage + Font Awesome 6.0

## 🎯 功能模块详解

### 一、入口与认证模块

#### 1.1 欢迎页（index.html）
- 展示品牌 Logo 和标语「高品质拍立得相机租赁平台」
- 三大入口按钮：用户登录、用户注册、商户入口
- 自动检测登录状态：已登录用户自动跳转至首页
- 模拟手机顶部状态栏（时间、信号、Wi-Fi、电量）

#### 1.2 用户登录（login.html）
- **表单字段：** 手机号输入、密码输入（支持密码可见切换）
- **记住我功能：** 勾选后保持登录状态
- **忘记密码链接：** 引导至密码重置流程
- **第三方登录：** 微信、QQ、微博快捷登录按钮
- **表单验证：** 实时验证输入格式，空值检测，错误提示
- **认证流程：** 调用 `auth.login()` → 存储 localStorage → 跳转首页

#### 1.3 用户注册（register.html）
- **表单字段：** 手机号、短信验证码（含发送验证码按钮）、设置密码（6-20位）、确认密码
- **密码强度验证：** 两次密码一致性校验
- **服务条款：** 需勾选同意《服务协议》和《隐私政策》方可注册
- **注册流程：** 调用 `auth.register()` → 创建账号 → 跳转首页

## 💻 技术实现细节

### JavaScript 核心功能（assets/js/main.js）

#### 初始化系统
页面加载时自动初始化以下模块：
- `initBackButtons()` - 返回按钮导航
- `initNavigation()` - 底部导航栏路由
- `initFormValidation()` - 表单验证引擎
- `initPasswordToggle()` - 密码可见切换
- `initLazyLoading()` - 图片懒加载（IntersectionObserver）
- `initTouchFeedback()` - 触摸反馈效果
- `initTabSwitching()` - Tab 切换系统
- `initCarousel()` - 轮播图引擎
- `initSearch()` - 实时搜索系统
- `checkDarkMode()` - 暗黑模式检测

#### 认证模块（auth 对象）
| 方法                        | 功能         | 数据操作          |
| --------------------------- | ------------ | ----------------- |
| `auth.login(user, pass)`    | 登录验证     | 写入 localStorage |
| `auth.register(user, pass)` | 用户注册     | 创建用户记录      |
| `auth.isLoggedIn()`         | 登录状态检查 | 读取 localStorage |
| `auth.getUsername()`        | 获取用户名   | 读取 localStorage |
| `auth.logout()`             | 退出登录     | 清除 localStorage |
| `auth.checkAuth()`          | 鉴权守卫     | 未登录重定向      |

#### 数据持久化
- 使用 `localStorage` 存储用户状态
- 存储键值：`user_logged_in`、`username`、`login_time`、`register_time`
- 无后端 API 调用，纯前端模拟

### CSS 设计体系（assets/css/style.css）

#### 设计规范
- **主色调：** #EC4899（粉色）+ #EF4444（红色）渐变
- **字体：** PingFang SC, Helvetica Neue, Arial
- **圆角：** 12px（卡片）、9999px（按钮）、40px（容器）
- **阴影：** 0 4px 12px rgba(0,0,0,0.08)

#### 响应式方案
- 默认画布：390px × 844px（iPhone 尺寸）
- 断点：@media (max-width: 400px) 全屏适配
- Sticky 头部和底部导航

#### 动画效果
- 页面过渡：0.3s 淡入动画
- 卡片悬停：上移5px + 阴影加深
- 骨架屏加载：Shimmer 闪烁效果
- 触摸反馈：涟漪效果

#### 暗黑模式
- 自动跟随系统 `prefers-color-scheme: dark`
- 深色背景 (#1f2937) + 浅色文字 (#f3f4f6)

## 🔄 业务流程

### 用户租赁流程
```
浏览首页/分类 → 搜索产品 → 查看详情 → 选择租期/日期/配送
    ↓
下单支付（租金+押金）→ 等待发货 → 收货使用 → 归还设备
    ↓
平台检查 → 退还押金（3个工作日）→ 评价
```

### 订单生命周期
```
待付款 → 待发货 → 待归还 → 已完成
               ↘ 售后处理中
```

### 逾期处理
- 超时费率：日租金 × 1.5倍
- 每超过1天按此标准计费

## 🚀 如何运行

1. 直接在浏览器中打开 `index.html`
2. 或使用任意 HTTP 服务器：
   ```bash
   # Python
   python -m http.server 8080

   # Node.js
   npx serve .
   ```
3. 在浏览器中访问 `http://localhost:8080`
4. 推荐使用 Chrome DevTools 的移动端模拟模式（iPhone 12/13 尺寸）
